    # 卡片组件

    # 案例展示

    苏州市乐营房车酒店营地两天一夜含双早&活动套票&小清新露营嘻嘻嘻&小清新露&小清新露&小清新露～
    1,300,017 积分

    # 代码示例

    <re-Card imgSrc="test.png" :width="226" :imgHeight="148" summary="苏州市乐营房车酒店营地两天一夜含双早&活动套票&小清新露营嘻嘻嘻～">
  <template v-slot:footer>
    <div class="footer">
      <span>1,300,017</span>
      <em>积分</em>
    </div>
  </template>
</re-Card>

    # Attributes

    参数 说明 类型 是否必填 默认值
    width 卡片宽度 Number false -
    imgSrc 图片地址 String true -
    imgHeight 图片高度 Number false -
    Summary 概述描述信息 String/slot false -
    footer 卡片底部 slot false -
